# 产品需求文档 (PRD)：macOS 极简专注时刻 (Focus Time)

## 1. 文档概述
* **项目名称**：专注时刻 (Focus Time)
* **文件版本**：V2.0（最终完善版）
* **创建日期**：2026年5月16日

---

## 2. 产品定位与核心价值
本应用是一款专为 macOS 设计的极简倒计时专注工具。它融合了 Apple 高端视觉美学与液态玻璃 (Liquid Glass) 质感，采用“纯数字极简”与“毛玻璃卡片”的混合布局。
* **核心价值**：在无操作时提供“零干扰”的绝对专注环境；在交互时提供高流畅度、具备呼吸感与空间深度的视觉愉悦感。
* **设计哲学**：克制。液态玻璃质感仅作为交互触发表层或特定数据区块的容器，不进行全页面无差别铺满，保留大量的桌面呼吸空间。

---

## 3. 全局设计规范与视觉资产
* **设计风格**：新中式留白感与 Apple 高端极简主义的融合。
* **核心材质**：macOS 原生液态玻璃材质（自适应模糊、高光边缘、1px 柔和微边框，融入系统壁纸背景）。
* **字体规范**：系统默认字体（SF Pro / Helvetica Neue），数字部分采用等宽非衬线设计，防止倒计时跳动引起的视觉抖动。
* **色彩矩阵**：
  * 纯白 / 经典黑（基础文字）
  * 专注橙 (`#FF5722`) / 活力黄 / 盎然绿 / 邃空蓝 / 优雅紫（严格去除粉色等低对比度颜色，确保数字在玻璃材质上绝对清晰）
  * 按钮主色：高饱和度科技蓝 (#007AFF)

---

## 4. 功能需求详述

### 4.1 桌面纯数字状态（默认专注模式）
* **界面表现**：
  * 仅显示 `MM:SS` 格式的倒计时数字。
  * 窗口完全透明、无边框、无阴影、无任何应用级的背景干扰。
* **交互行为**：
  * **鼠标穿透**：未开启置顶时，默认不阻挡下方底层窗口的正常点击。
  * **状态转换**：当鼠标移入数字区域时，自动无缝淡入切换至“鼠标悬停效果”。

### 4.2 鼠标悬停状态（Hover 快速操作层）
* **界面表现**：
  * 纯数字外围自动浮现一层轻量的**液态玻璃圆角卡片**。
* **新增交互（快捷闭环）**：
  * **快速暂停/继续**：支持**双击纯数字区域**直接触发暂停或继续；数字下方透出轻量的“播放/暂停”小图标。
  * **重置按钮**：提供微型重置图标，快速恢复初始时间。

### 4.3 窗口拖拽与缩放机制
* **拖拽移动**：鼠标悬停卡片显现后，按住非热区可自由拖动。
* **边缘缩放 (Hitbox 判定)**：仅在“鼠标悬停卡片显现”或“面板展开”时允许拖拽边缘进行矢量平滑缩放。

### 4.4 设置面板（展开状态）
* **触发方式**：桌面模式下单单击纯数字或悬停卡片，平滑展开。
* **面板功能区块**：
  1. **状态置顶 (Pin)**：右上角“置顶”开关。
  2. **时间预设快选**：`25分钟`、`30分钟`、`45分钟`、`自定义` 四个高亮胶囊。
  3. **自定义滑块 (Slider)**：区间 「1分钟 - 90分钟」，滑块上方放大显示当前数值。
  4. **数字颜色选择器**：提供 5 种高对比度纯色圆形色块。
  5. **显示选项 (Display Mode)**：
    * 采用横向三段式透明胶囊按钮（Segmented Control）。
    * 选项包含：【仅菜单栏】 / 【仅桌面】 / 【同时显示】。
    * **交互表现**：当前选中项带有轻微高光背景块，切换时带有原生平滑滑动手感。此设计在物理层面上彻底规避了“全部隐藏”的非法状态。
  6. **底部控制核心**：【开始/暂停】（蓝色主按钮）、【重置】、【关闭面板】。

### 4.5 菜单栏图标与下拉菜单控制
* **驻留形态**：常驻于 macOS 右上角系统菜单栏。
* **视觉表现**：
  * **倒计时数字**：精简显示（如  24:30）。
  * **进度圆环**：紧邻数字旁，根据剩余时间平滑递减指示进度。
* **下拉菜单交互 (核心指令区)**：
  点击菜单栏图标，弹出下拉菜单，必须按顺序包含以下功能列表（层级间需用原生分割线 `<NSMenuItem.separator()>` 隔开）：
  1. **状态与时间展示**：显示当前状态（专注中/已暂停）与精确剩余时间（置灰不可点击状态）。
  2. **快捷预设**：25 分钟 / 30 分钟 / 45 分钟 / 自定义（点击唤起桌面设置面板）。
  3. **面板与控制**：`显示/隐藏设置面板` / `暂停/继续专注` / `重置时间`。
  4. **系统指令**：`开机自启动 (Toggle)` / `退出专注时刻`。

### 4.6 倒计时结束反馈机制 (Completion Behavior)
* **视觉表现**：时间到达 00:00 时，桌面纯数字呈现缓慢的呼吸闪烁状态，菜单栏圆环闭合闪烁。
* **听觉反馈**：伴随一声温和、清脆的系统级提示音（如 Glass 或 Ping），避免惊吓用户。
* **状态重置**：用户单击闪烁的数字或菜单栏图标，即刻停止闪烁，并自动重置回上一次设定的倒计时时长。

## 5. 边缘场景与快捷操作

### 5.1 专注中断快速响应
* **场景**：用户在专注期间突遇紧急打扰。
* **逻辑**：除双击数字暂停外，支持全局快捷键（推荐：`Option + Space` 或用户自定义）一键挂起/恢复计时，最大程度降低交互繁琐带来的挫败感。
